## Collapsed crumbs

When the trail grows too long, collapse the middle crumbs behind an ellipsis menu that opens the hidden crumbs.

<ComponentExample code={collapsedCrumbsSource}>
  <CollapsedCrumbsDemo client:visible />
</ComponentExample>

## Menu crumb

A crumb whose label opens a menu to switch the current step between sibling contexts without leaving the trail. Its own chevron is the breadcrumb chevron, so no separator follows it.

<ComponentExample code={menuCrumbSource}>
  <MenuCrumbDemo client:visible />
</ComponentExample>

## View switcher

The menu crumb composes propel's `Menu` rows, so it supports a single-select list — the active view carries `selected` and shows the trailing check.

<ComponentExample code={viewSwitcherSource}>
  <ViewSwitcherDemo client:visible />
</ComponentExample>
